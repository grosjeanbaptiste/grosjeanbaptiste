## 🏗️ Acteble Tech Stack

### Backend (Rust)
- **Rust** - Systems programming language for safety and performance
- **Tokio** - Asynchronous runtime for reliable I/O
- **Axum** - Web framework for building scalable APIs
- **SQLx** - Type-safe SQL query builder with compile-time verification
- **Event Bus** - Internal event-driven architecture for decoupled communication
- **Event Sourcing** - Planned implementation using PostgreSQL for auditability and temporal queries

### Mobile (Flutter)
- **Flutter** - Cross-platform UI toolkit for building natively compiled applications
- **Riverpod** - State management solution for predictable state management
- **slang** - Advanced internationalization (i18n) with compile-time type safety
- **Melos** - Workspace management for Dart/Flutter monorepos

### Architecture
- **Clean Architecture** with clear separation of concerns
- **Domain-Driven Design** with Bounded Contexts
- **Multi-crate/Package** structure for better maintainability
- **Asynchronous** processing for high throughput
- **Feature-based** organization for better scalability
